# 24点计算 App

一款简洁好用的24点益智游戏应用。

## 应用介绍

24点计算是一款经典的数学益智游戏，目标是通过加减乘除运算，将四个数字计算得到24。本应用专为Android平台开发，界面简洁，操作流畅。

## 功能特点

- **四种难度等级**
  - 入门：数字1-9，只用加、减、乘
  - 容易：数字1-9，加、减、乘、除
  - 中等：数字1-10
  - 困难：数字1-13

- **智能提示**：遇到困难时点击提示按钮，系统会显示正确的计算过程（不显示负数）

- **撤销功能**：支持撤销上一步操作，方便试错

- **难度记忆**：自动记住用户上次选择的难度

## 技术栈

- **开发语言**：Java
- **最小SDK**：24 (Android 7.0)
- **目标SDK**：34 (Android 14)
- **UI框架**：Material Design
